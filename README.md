# 価値観の旅

約200の言葉の中から、いまの自分を最も深く貫く **5つの言葉** を選び取るためのワークツールです。
選んだ言葉をもとに、AIに渡せる「自己分析プロンプト」が生成されます。

## 使い方

1. ブラウザで開く（公開URL：https://ta5899100-creator.github.io/value-journey/ ）
2. 表示される言葉を絞り込み、最終的に5つを選ぶ
3. 生成されたプロンプトをコピーして、お好みのAIに貼り付ける

## プライバシー

入力・選択した内容は **一切サーバーに保存されません**。すべてブラウザ内で完結します。

## 出典

本ツールは [word-journey.site](https://word-journey.site) のワークを原案としています。
